# PRO 17025 Chat

Одностраничное веб-приложение для структурированных обсуждений по темам, связанным с **ISO/IEC 17025** (лаборатории, персонал, оборудование, методики, риски и т.д.). Данные хранятся только в браузере (`localStorage`), сервер не требуется.

## Возможности

- Папки по разделам (предзаполнены пункты 6.2, 6.4, 7.2, 7.7, 8.5, 8.7, 8.9 и др.) и произвольные папки
- Сообщения с текстом, опциональной ссылкой и меткой выбранного файла (имя файла сохраняется; содержимое файла в хранилище не кладётся)
- Очистка чата в текущей папке, удаление папок (кроме последней)
- Адаптивная вёрстка для узких экранов
- Отправка по **Ctrl+Enter** в поле сообщения

## Запуск

### Вариант 1: файл

Откройте в браузере файл [`index.html`](index.html) (двойной щелчок или перетаскивание в окно браузера).

### Вариант 2: локальный сервер

Удобно, если браузер ограничивает доступ к `localStorage` при открытии с `file://`.

```bash
npm install
npm start
```

Затем откройте в браузере адрес, который покажет [serve](https://github.com/vercel/serve) (по умолчанию порт **3000**).

## Структура репозитория

Точка входа приложения — **`index.html`** (ранее проект мог храниться как `Про Чат.html`).

| Файл / папка | Назначение |
|--------------|------------|
| `index.html` | Приложение: разметка, стили, логика |
| `package.json` | Скрипт `start` для локального превью |
| `.gitignore` | Игнор служебных и артефактов сборки |

## GitHub Pages

В настройках репозитория: **Settings → Pages → Build and deployment → Source: Deploy from a branch**, ветка `main`, папка `/ (root)`. Сайт будет отдавать `index.html` из корня.

## Ограничения

- Данные привязаны к браузеру и устройству; резервная копия — экспорт вручную (при необходимости можно расширить проект экспортом/импортом JSON).
- Вложения: сохраняется только имя файла, не бинарное содержимое.
